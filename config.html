Colonnes Bujo
▼

La vue principale du Bujo affiche jusqu'à trois colonnes. Masquez celles dont vous n'avez pas besoin.

Agenda semaine
Colonne droite : vue 7 jours
Projets
Colonne latérale : projets actifs
Calendrier sidebar
Mini-calendrier dans la barre latérale
Future logs
Navigation mois / années dans la sidebar
🎨
Couleur d'accentuation
▼

Définit la couleur principale du widget (boutons, badges, bordures actives…). Choisissez dans la palette ou entrez un code hexadécimal.

Laissez vide pour conserver la couleur par défaut (ambre #e8a020).
📖
Tous les paramètres
▼
Paramètre Valeurs Effet
titre texte libre Remplace le label « Bujo » dans l'onglet et le titre de page
accent #RRGGBB ou RRGGBB Couleur d'accentuation globale (boutons, bordures actives, badges)
lectures off Masque l'onglet Lectures
collections off Masque l'onglet Collections
tab bujo · lectures · collections Onglet ouvert au chargement
colweek off Masque la colonne « Agenda semaine » du Bujo
colprojets off Masque la colonne « Projets » du Bujo
colcal off Masque le mini-calendrier de la sidebar Bujo
colfuture off Masque la section « Future logs » de la sidebar
bday off Masque la gestion des anniversaires (bouton, onglet, affichage)

Note d'implémentation : Les paramètres colweek, colprojets, colcal, colfuture et tab nécessitent l'ajout de quelques lignes dans la fonction applyConfig() de votre index.html. Le snippet est fourni ci-dessous.

Snippet à ajouter dans applyConfig()
  // Onglet par défaut
  const tabParam = p.get('tab');
  if (tabParam && ['bujo','lectures','collections'].includes(tabParam)) {
    document.addEventListener('DOMContentLoaded', () => switchTab(tabParam));
  }

  // Colonnes Bujo
  if (p.get('colweek') === 'off') {
    document.addEventListener('DOMContentLoaded', () => {
      const el = document.getElementById('bujoWeekCol');
      if (el) el.style.display = 'none';
    });
  }
  if (p.get('colprojets') === 'off') {
    document.addEventListener('DOMContentLoaded', () => {
      const el = document.getElementById('bujoProjetCol');
      if (el) el.style.display = 'none';
    });
  }
  if (p.get('colcal') === 'off') {
    document.addEventListener('DOMContentLoaded', () => {
      const el = document.querySelector('.cal-wrap');
      if (el) el.style.display = 'none';
    });
  }
  if (p.get('colfuture') === 'off') {
    document.addEventListener('DOMContentLoaded', () => {
      const el = document.getElementById('futureLogs');
      if (el) el.style.display = 'none';
      // Masquer aussi le nav-div au-dessus si futureLogs est vide
      const nav = el ? el.previousElementSibling : null;
      if (nav && nav.classList.contains('nav-div')) nav.style.display = 'none';
    });
  }
